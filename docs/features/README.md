## Core Features

This boilerplate includes:

### Content Management
- ✅ MDX articles with frontmatter
- ✅ Multi-language support
- ✅ Static generation
- ✅ SEO optimization

### Authentication
- ✅ Auth.js (NextAuth.js) integration
- ✅ Multiple providers (GitHub, Google, Email/Password)
- ✅ Session management
- ✅ Protected routes

### Database
- ✅ DrizzleORM
- ✅ PGlite for local development
- ✅ PostgreSQL for production
- ✅ Type-safe queries

### Styling
- ✅ Tailwind CSS 4
- ✅ Custom theming system
- ✅ CSS custom properties
- ✅ Responsive design

### Developer Experience
- ✅ TypeScript
- ✅ ESLint & Prettier
- ✅ Git hooks with Lefthook
- ✅ Testing (Vitest, Playwright)
- ✅ Storybook

### Monitoring & Analytics
- ✅ Sentry error tracking
- ✅ PostHog analytics
- ✅ Better Stack logging
- ✅ Checkly monitoring

### Security
- ✅ Arcjet protection
- ✅ Rate limiting
- ✅ Bot detection
- ✅ Attack protection
